# ミャクミャクなりきりARアプリ 

文化祭の展示用に開発した、AI顔認識によるAR変身アプリです。

##  主な機能
- **リアルタイム顔認識**: MediaPipeを使用して顔を自動追跡します。
- **動的変身機能**: 映っている人数に合わせて、キャラクターの色が変化します。
- **アニメーション**: ランダムな瞬きなど、生きているような演出を実装。

##  技術仕様書 (Webサイト)
開発のこだわりや内部ロジックの詳細は、以下のWebサイトで公開しています。
[オンライン仕様書はこちら](https://chi2456.github.io/Face-in-hole/)

##  使用技術
- Python 3.x
- OpenCV / MediaPipe
- Pygame (サウンド再生)
- Sphinx (ドキュメント生成)
